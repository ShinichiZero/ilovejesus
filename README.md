# I Love Jesus ✝

A sacred website dedicated to the King of Kings: **Jesus Christ**.

Built with the "Sacred Modernism" theme — pure white with Cartier Gold (#E3C28E).

## Features

- **Intro Gate** — White canvas with gold shimmer, dedication text, and light-wash entrance transition
- **Home Hub** — Central landing page with navigation to all sections
- **Life Page** — Orthodox vertical timeline covering the Annunciation, Baptism (Theophany), Ministry & Transfiguration, Passion & Crucifixion, and Holy Resurrection & Ascension
- **Miracles on Earth** — Grid of all 37 miracles with full Orthodox Study Bible (OSB) quotes
- **Multi-language** — English, Spanish, and Latin via react-i18next
- **Animated page transitions** — Framer Motion with AnimatePresence

## Tech Stack

- [Vite](https://vite.dev) + [React](https://react.dev)
- [Tailwind CSS v4](https://tailwindcss.com)
- [Framer Motion](https://motion.dev)
- [React Router v7](https://reactrouter.com)
- [react-i18next](https://react.i18next.com) + i18next

## Development

```bash
npm install
npm run dev
```

## Deployment

Automatically deployed to [GitHub Pages](https://shinichizero.github.io/ilovejesus/) on every push to `main` via GitHub Actions.
